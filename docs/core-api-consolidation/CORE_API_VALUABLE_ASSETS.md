## Disposition legend

- **MERGE-AS-IS** — already in-repo on the deployed branch; bring into `main` via merge, unchanged.
- **PORT** — needs manual reconciliation on merge (a file that also changed on main).
- **REWRITE** — keep the capability but the implementation needs rework before pilot.
- **DISCARD** — do not carry forward.
