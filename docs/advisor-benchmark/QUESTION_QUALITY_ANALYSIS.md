## Read — this is the ONE criterion LifeNavigator wins

The P0 question-quality work paid off in a measurable, real edge:

- **LifeNavigator** asks exactly one sharp, specific, decision-relevant question and makes it the point of the turn. When the correct advisory move is genuinely "diagnose before prescribing" (multi-goal prioritization, ambiguous promotions, irregular-income planning), LN's question is the best of the three.
- **Claude** often asks good questions too (6.6) — but **buries them** at the end of a long answer, after it has already advised, so the question does less work.
- **ChatGPT** frequently asks **no** question at all (3.7), or poses rhetorical ones ("Do you enjoy coaching people?") rather than eliciting a decisive input.

## The catch

LN wins question quality but the _win is hollow in context_: it wins because asking is **all it does**. In `fin-03` the judge noted LN "asks the single best diagnostic question but never advises." A great question with no follow-through still loses the scenario (LN won 0/50). The value of a question is realized only when paired with analysis the user can act on.

## Implication

Keep the question engine — it's a genuine asset. But a question must become **step one of a two-step turn** (sharp question _+_ the framing/analysis around it), not the entire turn. See `ACTIONABILITY_ANALYSIS.md` and `ADVISOR_V2_ROADMAP.md`.
