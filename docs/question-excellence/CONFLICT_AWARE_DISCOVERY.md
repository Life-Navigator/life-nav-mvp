# Conflict-Aware Discovery (Phase 4)

**Date:** 2026-06-16 · When goals compete, ask the tradeoff — not for information.

## Behavior

For active multi-pursuit narratives (family/career/legacy) with ≥2 competing goals, `narrative_question()` poses a concrete tradeoff that names the user's own goals and the contested resource (time, money, energy):

> "{goal A}, {goal B}, and {goal C} all seem to be converging over the next year or two — and they're competing for the same time, money, and energy. If one had to move more slowly so the others could succeed, which would be easiest for you to postpone?"

Goals come from `_competing_goal_labels` (concrete goals preferred over context/feelings). Structured tradeoffs also available from `LifeDiscoveryService._CONFLICTS` (e.g. family↔FI = money, career↔family = time).

## Live evidence (v118)

- **Family:** "…competing for the same time, money, and energy. If one had to move more slowly so the others could succeed, which would be easiest to postpone?" ✅
- **Career:** "Which move would open the most doors over the next two years — the role, the credential, or the network?" ✅ (paths compared)
- **Founder:** "Which of these would feel like the biggest regret ten years from now if it didn't happen?" ✅

The tradeoff is **suppressed** for burnout/crisis narratives (it would be tone-deaf) in favor of a balance/relief question — see CONSTRAINT_AWARE_DISCOVERY + EMOTIONAL_DISCOVERY.
