ARCHITECTURE SUMMARY
==============================================================================

Database:     Supabase Postgres (14 new tables) + Storage (2 buckets)
API Pattern:  REST routes (/api/scenario-lab/*) - Next.js App Router
Job System:   Database-backed queue (scenario_jobs table)
Security:     RLS on all tables, signed URLs (60min), auto-redaction
Feature Flag: FEATURE_SCENARIO_LAB_ENABLED in .env

Key Features:
• Manual inputs + document uploads → OCR extraction
• Scenario versioning (immutable save states)
• Monte Carlo simulation (10k iterations, <2s)
• Explainable outputs (top 3 drivers + top 3 risks per goal)
• Roadmap generation on commit
• PDF report generation
• Dashboard pin widget for one goal
