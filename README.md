# 🧠 Life Navigator Agents

Hierarchical multi-agent system with dual GraphRAG.

## Quick Start
```bash
# Install
python3.12 -m venv venv
source venv/bin/activate
pip install -e ".[dev]"

# Run
python main.py
```
