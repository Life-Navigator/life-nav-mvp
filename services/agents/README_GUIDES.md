## ⚡ Quick Start

### Option 1: Connect Your Frontend (localhost:3000)
**NEW!** Your frontend can now connect to the backend API!

1. **Start Backend**: `./START_MAVERICK_QUICKSTART.sh`
2. **Read**: [`FRONTEND_READY.md`](FRONTEND_READY.md) or [`docs/CONNECT_FRONTEND_QUICKSTART.md`](docs/CONNECT_FRONTEND_QUICKSTART.md)
3. **Test**: Open [`docs/test-api.html`](docs/test-api.html) in browser
4. **Integrate**: Copy API client code to your frontend

### Option 2: Use Command Line Chat
1. **Start Backend**: `./START_MAVERICK_QUICKSTART.sh`
2. **Read**: [`docs/guides/START_HERE.md`](docs/guides/START_HERE.md)
3. **Chat**: `./venv/bin/python3 chat_with_agent.py`
