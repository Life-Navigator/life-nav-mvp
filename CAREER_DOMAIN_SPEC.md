## 3. Backend endpoints (`domains/career.py`)

```
GET  /v1/career/summary  /profile  /skills  /compensation  /opportunities  /recommendations
POST /v1/career/profile  /goal  /skill  /experience
```

`skills` returns skill levels + gaps vs `job_targets`; `compensation` returns history + market-band positioning; `opportunities` returns ranked target roles + networking plan.
