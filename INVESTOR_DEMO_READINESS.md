# INVESTOR_DEMO_READINESS.md

**LifeNavigator Elite Pilot Validation Sprint** · supersedes prior versions.
Grounded in **live Playwright validation against prod** (ELITE_PILOT_VALIDATION.md), real onboarded users. The question: can we run an investor demo today, and on what path?

## Demo-able TODAY (strong, verified live)

- **Family Office** — the strongest single story. "If something happens to me, what happens to my family?" answered with readiness, 5 estate/trust/beneficiary/survivor/legacy pillars, household, real $ figures, risks + recommendations. Investor-grade.
- **Health Intelligence** — real labs vs reference ranges, supplements, medications, fitness, nutrition, action items.
- **Document trust** — upload → "Recently learned about you" with provenance + honest "pending confirmation" (now currency-formatted + dark-mode legible).
- **Reports** — six branded, generatable report types (Full Life, Financial, Compensation, Family & Protection, Decision, Education).
- **Finance** — fixed last sprint (dark mode, transactions-all, spending verified).

## Would STALL a live demo (must fix first)

1. **Recommendations page is blank** — clicking "Recommendations" shows nothing, even though the user has 9 recommendations. Embarrassing on a core surface.
2. **Advisor stalls** — "Relationship Manager is thinking…" 12s+ with no reply. The advisor is the headline of the pitch; a stall here is the worst place to fail. Also branded "Relationship Manager," not "Arcana."

## Recommended demo posture

- **Today:** curated golden path — Dashboard → **Family Office** → upload a document → **Reports**. Avoid clicking Recommendations and avoid a live advisor Q&A until #1/#2 are fixed.
- **After the top-2 fixes (low/med complexity):** end-to-end demo becomes safe — "ask Arcana → grounded answer → see recommendations → trace evidence → generate a report."

## Pre-demo must-fix

1. Recommendations render (blank → show the user's real recs).
2. Advisor responsiveness (streaming/progress + faster first token) + Arcana branding.
3. (Polish) Dashboard "still forming" → reflect known facts; surface Career comp card.

## Verdict

**Demo-ready on a curated path now; broadly demo-ready after the top-2 fixes.** The moat (Family, Health, documents, reports) already demos as premium; the risk is exactly the two surfaces a curious investor clicks (Recommendations, Advisor).
