## Education-specific notes

- **E7 (PDF) is unique to Education** — Career/Family don't need a report generator.
- **E4/E5 depend on Career** (compensation lift via Program→JobTarget) — that's why Career
  precedes Education in Phase 3.
- Catalog data (schools/programs/outcomes) is **central reference, cited by value** (E0
  decision) — never fabricated, always `source`+`as_of`+`confidence`.
- High-debt / law-school / licensure paths carry an escalating `education_guidance` boundary
  (→ financial advisor / admissions counselor).
